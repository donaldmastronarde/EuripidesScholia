List of Files in zip archives for release 1.1:

Release110xml.zip contains the XML-related files:

  • OrestesScholia20210419.xml (or similar name with a later date element) is the large XML file of the Orestes arguments and scholia through line 500.
  • teiScholiaSchema2020.rng is the Relax NG schema file for the preceding.
  • triTreatises2020.xml is the XML file for the three short metrical treatises that precede the triad plays in Tricliniusâ€™ autograph copy T.
  • triTreatisesModel.rng is the Relax NG schema file for the preceding.
  • manuscripts20210420.xml is the XML file for the manuscript listing.
  • manuscripts20201228.rng is the Relax NG schema file for the preceding.
  • scholiaAll2020.xsl is the file for XSLT processing to produce HTML of the complete set of scholia along with the arguments.
  • scholiaGlossOnly2020.xsl is the file for XSLT processing to produce HTML of the glosses alone, without the exegetic and paraphrase scholia.
  • scholiaMosch2020.xsl is the file for XSLT processing to produce HTML of the Moschopulean scholia (with the few Planudean included).
  • scholiaMoschThomTri2020.xsl is the file for XSLT processing to produce HTML of the Moschopulean (and Planudean), Thoman, and Triclinian scholia.
  • scholiaNoGloss2020.xsl is the file for XSLT processing to produce HTML of the set excluding glosses of all types (also excluded are diagrams and word order annotations).
  • scholiaRec2020.xsl is the file for XSLT processing to produce HTML of the set excluding the old scholia.
  • scholiaThom2020.xsl is the file for XSLT processing to produce HTML of the Thoman scholia.
  • scholiaTri2020.xsl is the file for XSLT processing to produce HTML of the Triclinian scholia.
  • scholiaVetOnly2020.xsl is the file for XSLT processing to produce HTML of the old scholia.
  • the following subsidiary files that are referenced in include statements within the previous files:
    • linksAndActions.xsl
    • match-bibl2020.xsl
    • match-div4s2020june.xsl
    • match-segs2020.xsl
  • triTreatises.xsl is the file for XSLT processing to produce HTML of the Triclinian treatises.
  • manuscripts2021.xsl is the file for XSLT processing to produce HTML of the Manuscripts page.
